# Food-donation
Dự án cá nhân - Ứng dụng web kết nối các nhà hảo tâm và người cần nhận sự hỗ trợ từ cộng đồng
* Front-end: Typescript, Material UI
* Back-end: NestJS
* Database: MongoDB Cloud

## Getting Started
Start project with 2 terminal:
```
cd ./frontend
npm run start
```

```
cd ./backend
npm run dev
```
